# PVZ2国服插件

- 可配置模块 - 友好的配置界面
- 支持大多数版本（包括后期更新的版本）
- 依赖 LSPosed / Xposed / EdXposed / VirtualXposed

### 功能

- [x] 解锁帧率上限 -- 可调节 30-60-90 帧
- [x] 修改游戏设备ID -- 自由切换设备
- [x] 广告10秒 -- 仅限TapTap版

### 测试环境

- 雷电模拟器9 + LSPosed + PVZ2-TapTap-v3.1.5
